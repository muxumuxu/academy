    • Ce qu’est le code, le HTML et le CSS, et comment ces 2 langages fonctionnent ensemble.
    • Comment construire ton premier site Internet et le mettre à jour quand tu veux.

  • À qui s'adresse ce cours

    • Aux débutants et néophytes du code, qui souhaitent apprendre à construire un site Internet.
    • A tous les créatifs, qu’ils travaillent dans le design ou le marketing.
    • Aux personnes qui souhaitent s’initier au code et comprendre comment le développement informatique fonctionne.
